Привет, меня зовут Андрей.

Я инженер в ИТ, который любит изучать новые технологии и полюбил делиться своими знаниями с другими.

Начал с Java,
  * [DLTK](https://github.com/eclipse-dltk) => [DLTK на Eclipse.org](https://www.eclipse.org/dltk/)
  * [RCPTT](https://github.com/eclipse-rcptt) => [RCPTT на Eclipse.org](https://www.eclipse.org/rcptt/)
